5.12思想总结

Posted on 2019-05-12 | In Emotional(情感)

​ 前一段时间因为自己心性的不成熟，而萌生了一些比较不好的念头，在经过老师的指导和教诲之后，也确实认识到了自己的不足和自己处于这个年龄段的劣势。

​ 1.选择逃避，而不去思考如何改变自己。进入五月之后，大家的状态都开始低迷了起来，看着大家这个样子，我的心里难免会多想一些事情，比如这种负面状态在公司的传播肯定会越来越快，时间久了，这种状态的创业肯定不会成功，既然不成功，为什么还要待在这里，与其浪费时间，不如趁早离开。这种想法在我的脑海里一直在驱使着我，让我离开。明明发现了问题，第一时间不是想着怎么去解决，而是想着如何去逃避、如何换一个环境让自己变得安逸，这种想法，是很可笑的。一个人在面对困难问题时，如果不是想着如何解决，而是想着如何逃避，这样只会让自己变得更加软弱，更加不成熟。就像现在，明知道团队状态低迷，不想着如何带动大家的情绪，反而在谋划着如何离开，如何去逃避这个现实，这就是一种软弱的表现。在和老师的交流中，老师很明确的就指出了我的错误，我选择离开，当时的情况看来，我还以为是一种很明智的选择；可现在想想，确实是很幼稚，也很可笑。

与其选择逃避，不如想想怎么通过努力去改变现在的状态，勇敢的面对现实。

​ 2.急功近利，无法看到远处的风景。前一段时间回学校的次数比较多，看见身边的同学选择考研的已经在紧锣密鼓的筹备中，也有一些比较厉害的同学已经拿到了大公司的offer，相比之下，看着自己那一身油渍的衣服，心里很不是滋味，于是就想，同样二十出头年纪，人家以后会有研究生的头衔，会有高额的工资，而自己却还是在饭店里焦头烂额，在体育场挥汗如雨。一番比较下来，就觉得自己不应该在这个年纪里如此狼狈，我也想像其他同学那样光鲜亮丽。于是，考研的想法在我的心里悄悄发芽，然后慢慢生长，最终，伴随着在这边的迷茫，终于还是爆发了。在和老师交流过后，终于发现了自己一个很大的问题—-急功近利。人生就好像一座山，山脚、山腰和山顶的风光也各有差异，山脚的环境舒适，花草众多，风景优美；而山腰因为海拔偏高，地势险峻，风景也大不如山脚；山顶环境则是更加恶劣，无花无草，然而眺望远方，却又有山脚和山腰领略不到的风景，崇山峻岭，连绵不断，大江大河，气势磅礴。进入大学，我们就好似开始登山一般，现在的我们，就好像被老师带到了半山腰，其实接下来只要再努力，就可以爬到山顶，一览众山小，而我当时的状态就是，走到了半山腰，发现很多人在山脚纷纷驻足，无心登山，而我自己也渐渐被山脚的风景所吸引，又考虑到山腰的环境恶劣，地势险峻，于是就想回到山脚，去欣赏山脚的风景，却全然忘记了自己已然比他们更高一层，更可笑的是，现在竟然想着怎么再回到原点！

所以，不要被眼前的利益蒙蔽了双眼，我们一定要学会往前看，要学会站在客观的角度看问题。所以我们只需要坚定不移的往下走就可以了，坚持体育训练，坚持饭店的经营，总有一天，我们会在他们触碰不到的高度去俯视他们。

​ 3.个人主义，然而这个社会需要团队。当今社会，一个人单打独斗是不可能成功的，毕竟人无完人，一个人不可能是完美的，他有长处肯定也会有短处，在社会的大熔炉之中，肯定会无限放大他的短处，从而慢慢导致失败。然而团队不一样，团队成员虽各有短处，但相互之间可以取长补短，互帮互助，这样一来，虽然团队成员不是完美的，但是这个团队就会变得接近完美。所以很多时候我们要摒弃掉自己的个人主义，在看见团队的问题时，要先从自身找原因，先从自身做起，而不是去想别人做的怎么样，去指责别人。可能现在这个年纪的我们因为考虑问题不全面等原因，经常会在权衡个人意识和团队意识时，偏向于个人意识，但是只有当你试着放下个人主义，尝试改变自己、适应团队的时候，才是一个人真正成长的时候。

我们是一个团队，所以凡事都要以团队为单位考虑问题，不要遇见什么事都先想自己，无论是好事占便宜，或者是坏事往自己身上揽，这些都是错误，都需要改正，不会再让这种事发生在团队中间。

​ 4.不要怀疑，要相信老师。以前总想着所有的东西在亲眼所见之前，都是不准确的，就好像现在老师和师哥整天说的社会，这个社会很残酷也很现实，可那终究是他们眼中的社会，或许是自己还对这个社会抱有一丝幻想，于是就想亲眼看看这个社会，去亲身感受一下这个社会。其实这样摒弃掉团队，去自己闯荡，最后的后果并不是会一个好结果。就好像现在前面有一棵树，你闭着眼什么也不知道，老师说，前面有一棵树，别走了，但是你就是想一头撞上去，去证明那是棵树，但是呢，你的确知道那是棵树了，树不会吃亏，可你不疼吗？老师经历的事情很多，并且老师也是发自内心的想要培养我们，师者，传道受业解惑，老师确实已经将“老师”这个角色做到了最好，有这样的老师，竟然不去相信，现在想想，确实很可笑。

​ 以后肯定会坚定不移的相信老师，不再对这个团队抱有怀疑态度了，只有当你真正相信别人时，别人才会彻底相信你。

​ 所以，以后情绪再低落的时候，与其想要逃避，不如从自己身上找原因，逃避问题永远不会解决问题，只有勇敢面对才能解决问题。数字从一至十，虽然只有一竖之差，而中间却要经过九层磨炼，每一层都是成长，每一层都是历练，文字如此，更何况人呢。

hexo生成博文时，图片不显示问题

Posted on 2019-04-08 | In Hexo

​ 在写上一篇文章时，突然踩到一个坑，因为是第一次在hexo博文里加图片，第一次嘛，毕竟没经验，所以截好图之后直接就在typora里Ctrl+v了=-=，在md文件里还能显示图片，但是当我写完然后hexo clean hexo d -g之后，再看我的博客，居然显示不出来！

​ 于是一通百度之后，终于找到了解决办法：

​ 1.切换到你的博客目录，然后右键空白的地方->git bash here

​ 2.输入下面的命令，安装一个图片插件

​ npm install hexo-asset-image --save

​ 3.这个时候，当你再hexo new xxx时，就会在_post里生成一个.md和一个同名的文件夹，你就可以把你的图片放在文件夹里

​ 4.如果之前你已经hexo new xxx了，那就直接在_post文件夹里新建一个xxx文件夹就ok啦

​ 5.注意当你写博文的时候，图片的路径不能用绝对路径，一定要用相对路径！！

​ 4.然后写好博文之后hexo clean hexo d -g，刷新一下你的博客，就会出现你的图片啦！

windows下的redis自启动

Redis下载

下载地址：https://github.com/MSOpenTech/redis/releases。

根据自己的电脑选择是32位的还是64位的

然后下载、解压

Redis开机启动

1.在redis的目录下执行（执行后就作为windows服务了）

redis-server --service-install redis.windows.conf

2.然后手动启动redis

redis-server --service-start

3.停止服务

redis-server --service-start

4.卸载服务

redis-server --service-uninstall

随之而来的压力

​ 不知不觉，已经在龙代待了100多天了，这么久了，让我感触最深也是最能直接感受到的就是自己身体的变化，从之前的五千米都跑不下来，到现在五千米能够跑进20分钟，对自己来说，是一种挑战，也是一种改变。

​ 然而随之而来的，也是种种的压力，看着自己遇到瓶颈而无法突破的迷茫，看着队友一直在进步而即将超越自己的压力，听着老师对自己的种种期盼而无能为力的无奈，眼瞅着运动会即将到来而却因为自己的懒惰一次次懈怠训练，昨晚开会刚说公司要进行文化宣传，这次的运动会如果能够取得一个好的成绩，一定会是一个对公司最大的宣传，而自己却······种种的一切让自己不知所措，如果说在这个团队自己的亮点是跑的快，那这个亮点眼看着暗淡了下来，自己也越来越找不到自己的位置，迷茫、惊慌、无奈、无助充斥着自己的内心。

​ 有些人是靠天分，有些人是靠努力，而不断努力的天才也大有人在。如果说自己真的是靠天分在运动方面领先他人，那那些努力的人是不是太被我小瞧了呢。是时候正视自己了，不能再凭借着自己的优越感而目空一切、眼高手低了。

​ 加油吧少年，相信你会变成那个肯努力的天才。

2018总结

Posted on 2019-03-30 | In Emotional(情感)

这博客的第一篇正式文章，就写一篇2018年的总结吧。

​ 曾经在公司一块出去吃饭的时候说过一句话，2018年，我失去了很多，不过也收获了很多。

失去

​ 先是奶奶去世，后来是在女朋友即将去韩国的时候分手（现在应该要改口叫前任了吧），然后在奶奶走后的第一百天，连时辰都一样，都是凌晨三点，爷爷也跟着去了。一百天的时间里，先后失去了三个最爱我的人。

​ 在后来的日子里，又重新开始抽烟，重新开始浪，似乎失恋对我来说更像是一种解放。刚开始一直以为可能自己并不是很爱她吧，要不然在分手之后也不会这么坦然，直到在知道了她有了新男朋友之后，那个时候，才真正明白心痛是什么感觉。真正认识到后悔的时候，不是分手之后，而是她有了新欢的时候，而那天，也是我真正哭了的一天。是啊，忘记一个人有两种方式，她选择了新欢，而我只能选择时间。

​ 奶奶走的那天，我刚坐上去往北京的快车。当我在济南得知奶奶走了的消息的是时候，我的心里甚至没有很大波澜，脑袋一片空白，因为奶奶已经92岁了，在床上躺了两年，走的时候也是平平静静，没有痛苦，所以我一直在不断的安慰自己，奶奶是去享福了，奶奶是去享福了···然后请假、下车、以最快的速度回家，见到奶奶的时候，奶奶已经在棺材里了，当我跪下的时候，我的眼泪终于还是没崩住，爷爷抱着我，一直在安慰我，甚至还扇我耳光，让我别哭了。其实我心里明白，爷爷才是最舍不得奶奶的，要不然也不会在那么热的天，守在奶奶的棺材旁，一句话也不说的坐了三天。没能见到奶奶的最后一面，是我迄今为止最遗憾的事。奶奶啊，你做的面条，真的很好吃啊，孙子真的还想再吃一次···

​ 爷爷走的时候，我是眼睁睁的看着爷爷慢慢失去了呼吸，那晚，全家人都没睡，爷爷也是很平静，大概是自己在床上躺够了吧，也或许是太想念奶奶了。我赶回家的时候，爷爷已经睁不开眼睛了，我叫了爷爷几声，想哭，缺又不敢哭，真的害怕这一哭，爷爷就真的离开我了。令全家人都很惊讶的是，在我叫了爷爷之后，爷爷竟然睁开眼看了看这个世界，我不知道是不是在寻找我，但是那个时候爷爷的眼睛已经没有了往日的色彩，并且眼神也没有了焦点，但是能感觉到出来他在寻找着什么。我使劲趴下去，靠近爷爷，想要让他看见我，但是爷爷的眼神告诉我，他已经看不到我了，我就只能使劲叫着他，让他听见我。似乎是因为见了爷爷最后一面的缘故，我的遗憾，相比于奶奶离开的时候，少了很多。但是爷爷走后的几天，每天晚上都格外的想他，想起以前一起和爷爷放羊、钓鱼、剥玉米，爷爷教给我卷烟，说以后要一直给他卷烟抽，还给我讲了很多他们那个年代的故事，那时候还小，不懂得珍惜，现在真的失去了，才明白当初在一起的时光有多么珍贵。爷爷啊，您什么时候再给我讲以前的故事啊，我保证不会再睡着了···

​ 奶奶九十二，爷爷九十八，两个人从桃李年华一直走过鲐背之年，相互嫌弃却又彼此珍惜。七十余载的人间烟火，换做天堂生活，相信您二老在天上也是最亮的星吧。

收获

​ 2018，转眼间就已经大三了哎，想想大一大二似乎所做的一切，都是在维系一段感情，万万没想到的是，那段感情离开了我，两年多的时间，似乎什么也没有得到。本想毕业以后培训机构、北漂或是考研，不了了之这大学四年，但是直到遇见赵老师、师哥师姐、还有我的八个小伙伴。虽然仅仅半年，但是确实对这个社会，这个大学有了一个新的认识，遇见这些人，至少在现在看来，是对的。

​ 六点起床，训练拳击。

​ 七点半，吃饭

​ 八点，上课

​ 十点，经营饭店

​ 三点，五公里+体能训练+娱乐

​ 晚上八点，学习按摩

​ 其他时间就是做项目

​ 每天都很充实，一天下来没有一刻是闲着的，也能明显的感觉到自己的身体、心理的变化，从最开始的五公里跑半小时到现在能跑进23min，从最开始的不敢和别人说话到现在可以侃侃而谈，从最开始的运行环境都不会配，到现在可以独自做一些项目。

​ 训练让我的身体变得更加健康，我也能感觉到自己身体的变化，至少现在不像以前那样看起来像个猴一样，平常走路也不想以前那样耸着肩驼着背，身体的各项机能也都有了很大的改变。

​ 如果说训练就像是打地基一样奠定我们的身体基础，那按摩就像是在健康的身体上改了一座大楼，让你的身体能够变得更加健康，因为学得不深，所以我就不过多说这方面了···

​ 抛弃了之前的迷茫，坚定了现在跟随老师好好做项目的决心，我相信我们一定会一步一步的变得更加优秀。
